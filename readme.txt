# üêç Snake Detection & Classification System

AI-powered system to detect snakes and classify them as venomous or non-venomous.

## Features
- Real-time snake detection using YOLOv8
- Venomous/Non-venomous classification using EfficientNetV2
- Batch image processing
- Live webcam detection
- Adjustable confidence thresholds

## Usage
Visit: https://snake-classification-detection-app.streamlit.app/

## Local Development
```bash
pip install -r requirements.txt
streamlit run app.py
```
